# Applesauce Loaders

A collection of observable based loaders built on top of [rx-nostr](https://penpenpng.github.io/rx-nostr/)

## Installation

:::code-group

```sh [npm]
npm install applesauce-factory
```

```sh [yarn]
yarn install applesauce-factory
```

```sh [pnpm]
pnpm install applesauce-factory
```

:::
