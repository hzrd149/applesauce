# Text content

The `applesauce-content` package provides tools for parsing and rendering the text `content` of nostr events.

## Installation

:::code-group

```sh [npm]
npm install applesauce-content
```

```sh [yarn]
yarn install applesauce-content
```

```sh [pnpm]
pnpm install applesauce-content
```

:::
