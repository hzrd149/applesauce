1. `new RumorStore()` — `extends EventStore<Rumor>`, verifier locked to `verifyRumor`.

3. Generic models — `timeline()` → `Rumor[]`, `replaceable()`, `filters()`, `getEvent()` return `Rumor`-typed (RUMOR-04; the Phase-2 WR-02/D-02 seam closure).
4. Kind-5 delete rumor removes matching stored rumor via generic `DeleteManager<E>` (RUMOR-05).
