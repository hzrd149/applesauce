<objective>
The single-source-of-truth spine of Phase 7. Make `material.channels` the ONLY source of channel key material — removing `ChannelMetadata.key`/`.epoch` entirely (breaking, but concord is unreleased) — so a keyless private channel derives nothing instead of silently deriving the public address (H07), edition JSON is never trusted for key material (H06), and a rekeyed channel's key is never stale (H08 first root cause / CHAN-05). This same atomic change rewrites the `foldControl` channel loop to pick edition fields explicitly (CHAN-04) and to enforce sticky channel-deletion terminality with heads pinning (CHAN-07).

Purpose: Collapse the H06/H07/H08 root class into one refactor and close CHAN-07's resurrection defect at the compaction boundary. Every behavioral fix lands with its hand-derived, spec-derived test (never a self-referential assertion).
Output: The refactored derivation/fold source plus spec-derived tests for CHAN-01, CHAN-03, CHAN-04, CHAN-07, and TEST-01 (both CORD-03 §1 branches + keyless-derives-nothing).
</objective>
