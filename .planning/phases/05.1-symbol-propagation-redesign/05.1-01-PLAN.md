<objective>
Establish the redesign mechanism that ALL 43 migrations depend on, before any site migrates. Move the three gift-wrap symbols into applesauce-core so `PRESERVE_EVENT_SYMBOLS` becomes a static, load-order-independent Set (scope point 1), delete the import-time `.add()` mutation in common, and add the carry-forward half to BOTH strip loops so a preserved symbol dropped by an operation's own internal spread is explicitly restored by the pipe (scope point 2).

Purpose: Once symbol writes become non-enumerable (Plans 07-13), an operation's own `{ ...draft }` spread drops them; the pipe — not write-site enumerability — must guarantee survival. This plan builds that guarantee.
Output: Core-owned gift-wrap symbols, a static preserve set, two carry-forward loops, and the mechanism + symbol-move changesets.
</objective>
